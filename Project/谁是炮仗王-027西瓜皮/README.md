# 谁是炮仗王?
## 游戏名字
> 谁是炮仗王?


## 安装和启动游戏

### 安装依赖

```bash
$ yarn
```

### 启动

```bash
$ yarn start
```

## 使用技术
1. react: 17.+
2. umijs: 3.5.20
3. svg
4. animejs: 3.2.1
5. mojs: 1.3.0
6. ts

### 技术介绍
1. 使用`react` + `umijs` 的框架作为项目框架
2. 使用`svg` 画出鞭炮
3. 使用`animejs`实现游戏动画
4. 使用`mojs`实现点击反馈
5. 使用`ts`语言编写

## 游戏规则
首先点击炮仗开始游戏，在规定时间内，点击一定数量的炮仗，如果你成功完成，则会出现掘金屏裂效果。当然规定的时间比较短，你可以点击头像增加时间。
